# India VCS Carbon Project Explorer

An interactive dashboard mapping 1,167 voluntary carbon market (VCS) projects registered in India.

**[Open the Explorer →](https://yourusername.github.io/india-vcm-explorer/)**

## What it shows
- Project locations mapped with category-coded dots
- State and district boundary overlays
- Filters by category, status, state, vintage year, and crediting period
- Per-project detail panel with credits issued, retired, and remaining
- Direct links to each project's Verra Registry page

## Data sources
- Pamela Quartson, Barbara K Haya, Tyler Bernard, Aline Abayo, Xinyun Rong, Ivy S So, Micah Elias. (2026). Voluntary Registry Offsets Database v2026-04, Berkeley Carbon Trading Project, University of California, Berkeley. Retrieved from: https://gspp.berkeley.edu/berkeley-carbon-trading-project/offsets-database. 
- [Verra VCS Registry API](https://registry.verra.org) — project attributes and coordinates
